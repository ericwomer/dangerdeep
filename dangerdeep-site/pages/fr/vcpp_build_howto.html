HOW-TO: Build DftD with Visual C++

This is a how-to build Danger From The Deep source using Visual C++ 2005 Express Edition for Windows

Requirements

  1. Install Visual C++ 2005 Express Edition ( available here )
  2. Install Microsoft Platform SDK ( available here )
  3. Configure Visual C++ to use the PSDK ( available here )
  4. Download & install SDL for VC++ ( available here )
  5. Download & install SDL_Image ( available here )
  6. Download & install SDL_Mixer ( available here )
  7. Download & install SDL_Net ( available here )
  8. Download & install FFTW3 ( available here )

Building

  1. Download latest source of Danger From The Deep from CVS repository ( instructions are available here )
  2. Unzip vc2005.zip and extract the VC++ solution files & projects ( zip is available here or from the CVS tree )
  3. Copy the unzipped vc2005 folder along side the danger deep src directory, if the source is in "c:\games\dangerdeep-0.2.0\src" then the vc2005 directory containing the solution and projects should be copied to "c:\games\dangerdeep-0.2.0\vc2005"
  4. Create two new user environment variables named DDINCLUDE and DDLIB (Control Panel -> System -> Env Variables).

    For DDINCLUDE set the value as:
    "drive:\path\to\SDL\include";"drive:\path\to\SDL_Image\include";"drive:\path\to\SDL_Mixer\include";"drive:\path\to\SDL_Net\include";"drive:\path\to\FFTW3\include";"drive:\path\to\gl\include";
    For example:
    "c:\ddbuild\SDL\include";"c:\ddbuild\SDL_Image\include";"c:\ddbuild\SDL_Mixer\include";"c:\ddbuild\SDL_Net\include";"c:\ddbuild\FFTW3";"C:\Program Files\Microsoft Platform SDK\Include\gl";

    For DDLIB set the value as:
    "drive:\path\to\SDL\lib";"drive:\path\to\SDL_Image\lib";"drive:\path\to\SDL_Mixer\lib";"drive:\path\to\SDL_Net\lib";"drive:\path\to\FFTW3\lib";
    For example:
    "c:\ddbuild\SDL\lib";"c:\ddbuild\SDL_Image\lib";"c:\ddbuild\SDL_Mixer\lib";"c:\ddbuild\SDL_Net\lib";"c:\ddbuild\FFTW3";
  5. Launch DDVC2005.sln and build the game executable's in release or debug mode
  6. Post build the executable's will be in vc2005\debug or vc2005\release folder.
  7. Copy the exes to the dangerdeep install dir and run the game :-)
