Dernières Nouvelles

La version 0.3.0 est disponible

L'équipe de Danger from the Deep est fière de vous annoncer la sortie de la version 0.3.0 (ALPHA). Parmi les nouvelles fonctionnalités, vous trouverez :

  • Utilisation de l'OpenGL2.0/GLSL1.1 pour un meilleur rendu visuel
  • Nouveau manuel utilisateur
  • Nouvelle gestion des ombres (nécessite une carte vidéo compatible GLSL1.1)
  • 4 nouveaux types de sous-marins : IIA, IIB, IIC, IID
  • De nouvelles livrées pour tous ces sous-marins
  • L'écran "Cabine du Commandant" inclut maintenant le carnet de bord ainsi que d'autres nouvelles fonctions
  • Un nouveau destroyer, classe Tribal, avec 5 livrées
  • Correction de bogues de modélisation
  • Meilleur rendu de l'eau grâce à l'utilisation de GLSL 1.1
  • Correction de bogues graphiques
  • Nouvelle fonte de caractères pour toute la partie textuelle du jeu
  • Amélioration du ciel nocturne
  • Le périscope fournit maintenant une image brouillée lorsqu'il est mouillé
  • Tous les modèles sont maintenant capables d'avoir des parties mobiles (tourelles, canons, etc)
  • Le tableau d'honneur fonctionne correctement
  • La programmation se fait maintenant en multithread
  • Ajout d'un nouveau système de rendu sonore
  • Amélioration de l'écran de gestion des torpilles
  • Ajout d'une "caméra de torpille" de base
  • Amélioration de l'implémentation de la brume diurne
  • Meilleure gestion des éclaboussures d'eau lors d'un impact
  • Utilisation de la programmation à base de VBOs et FBOs
  • Nombreuses corrections de bogues

Les exécutables, paquets d'installation et le code source sont disponibles sur la page Téléchargement.

Nouveaux paquets d'installation

De nouveaux paquets d'installation pour Windows et Linux sont disponibles. Ils contiennent toutes les données nécessaires au jeu. Il vous suffit de télécharger le fichier et de l'exécuter. Puisque toutes les données sont incluses, la taille des paquets s'en ressent (60 Mo). Mais d'un autre côté, l'installation est limitée à son plus strict nécessaire et vous pouvez lancer directement le jeu. Bon amusement !

Port pour Win32

Déposé par Thorsten Jordan le 07-juil-2003 à 04:44

Nous avons commencé à travailler sur le port Win32 (VC++6 minimum). La compilation est possible, et le résultat s'exécute mais il y a toujours des bugs. Le pointeur de souris n'est pas visible sur certains ordinateurs et certaines commandes claviers ne fonctionnent pas. MAIS DftD est jouable :)

Structure de fichiers :

Copier le répertoire de données dans le répertoire où se trouve dangerdeep.exe.

Pour lancer le jeu, vous avez besoin de plusieurs DLLs: sdl.dll, sdl_image.dll et les DLLs liées (jpeg.dll, png.dll, zlib.dll, ...) qui doivent placées dans le répertoire où se trouve l'exe. Cf. http://www.libsdl.org pour le téléchargement.

De plus, vous aurez besoin de msvcrt.dll et d'autres DLLs VC6. Aidez-vous de Google pour trouver ces DLLs. Ce problème de DLLs devrait être réglé dans une prochaine version. Nous pourrons sans doute bientôt vous fournir une version Win32 complète.

Comment créer une mission personnalisée

Déposé par Thorsten Jordan le 28-fev-2003 à 05:59

Il suffit d'ajouter une ligne dans le fichier data/missions/list indiquant le nom du fichier sans extension (par exemple "testship") et créer une nouvelle entrée dans le fichier langues (data/texts/*.text). Ceci n'est pas absolument nécessaire mais sans cela la mission ne sera pas visible dans le menu.
